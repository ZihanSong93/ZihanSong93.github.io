Library Deserts

If you have a library and a garden, you have everything you need.

--Cicero
This work tells a story about Los Angeles library deserts. There are still places couldn't get access to libraries, and some of them are with comparatively high poverty rate. I created this work using Arc Pro, Illustrator, ArcGIS Online, and Esri Story Map.
